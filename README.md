## Week 1: Data Acquisition

In this week, I collected historical stock price data for multiple companies using Python and the yfinance library.

* Selected a diversified portfolio (Banking, IT, FMCG, etc.)
* Downloaded time-series data from 2020 to 2025
* Used Adjusted Close prices for accurate analysis
* Handled missing values using forward fill
* Exported the cleaned dataset as a CSV file

This dataset will be used in the next phase for return, risk, and portfolio analysis.
